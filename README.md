# 💎 JSON ID Extractor Pro

![Version](https://img.shields.io/badge/version-2.0.0-purple.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Vanilla JS](https://img.shields.io/badge/javascript-vanilla-yellow.svg)

**JSON ID Extractor Pro** adalah Single Page Application (SPA) futuristik yang dirancang untuk pengembang dan data analis guna mengekstrak `account_id` dan `game_uid` dari tumpukan file JSON secara instan. Dibangun dengan fokus pada kecepatan, estetika *glassmorphism*, dan pengalaman pengguna mobile-first.



---

## ✨ Fitur Unggulan

- 🚀 **Deep Recursive Extraction**: Menemukan ID bahkan di dalam objek atau array yang bersarang dalam (nested).
- 📂 **Multi-File Support**: Upload banyak file sekaligus dengan sistem Drag & Drop.
- 💅 **Pretty Sort Algorithm**: Fitur cerdas untuk mengurutkan "ID Cantik" (Pola angka berulang, urutan, palindrom, dan angka spesial).
- 🔍 **Detail Inspector**: Klik ID mana pun untuk melihat lokasi persis (path) dan konteks data asalnya dalam format JSON yang rapi.
- 📱 **Responsive Design**: Tampilan dashboard profesional yang beradaptasi sempurna di desktop maupun smartphone.
- 📋 **Quick Actions**: Copy all, export ke `.txt`, dan pembersihan workspace dalam satu klik.

---

## 🛠️ Spesifikasi Teknis

Aplikasi ini murni menggunakan teknologi web standar tanpa dependensi eksternal (Zero Dependencies):

* **Frontend**: HTML5, CSS3 (Custom Grid & Flexbox)
* **Engine**: Vanilla JavaScript (ES6+)
* **UI Style**: Modern Dark Theme dengan efek Glass-morphism
* **Performance**: Algoritma pencarian rekursif yang dioptimalkan untuk file besar.

---

## 🚀 Cara Penggunaan

1.  **Clone atau Download**: Simpan file `index.html` ke komputer kamu.
2.  **Buka di Browser**: Klik kanan file dan pilih browser favorit (Chrome, Edge, atau Firefox).
3.  **Upload JSON**: Tarik file JSON kamu ke area "Workspace".
4.  **Ekstrak & Analisis**:
    * Gunakan **Pretty Sort** untuk menemukan ID unik.
    * Klik pada baris ID untuk membedah data di **Detail Inspector**.
    * Klik **Copy All** untuk menyalin hasil akhir.

---

## 📂 Struktur Data Target

Secara default, aplikasi mencari kunci berikut dalam file JSON Anda:
- `account_id`
- `game_uid`
